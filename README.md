我不要熱死	LSA-1071
=====
簡介
----
題目發想來源是散熱器，大致上我們使用的散熱器很少能隨溫度而改變轉動速度。因此我們設計的是當物體在X度的時候會自動開啓風扇扇熱和燈泡發亮，達到X度的時候因溫度太高而蜂鳴器會發出警告聲和燈泡閃爍。
